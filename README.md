== The Bean Team Espresso Machine ==

By: Andy Tran, Casey Wolfe, Eric Hou, Zachary Au

The goal of this project is to modify the gaggia classic espresso machine to have precision control for brewing.

This Repo contains the following files:
- Assets Folder: contains jpeg and pngs that display icons used in the OLED screen
- Menu.h
- OLEDMenu.ino
- bitmaps.h
- features.html
- hardware.html
- index.html
- menu.cpp
- pins.h
- pressure_control.cpp
- pressure_control.h
- pressure_sensor.cpp
- pressure_sensor.h
- psm.cpp
- psm.hpp
- tempControl.cpp
- tempControl.h
- temp_sensor.cpp
- temp_sensor.h
